# Popout - Find the Red Dot

A fast-paced visual search game inspired by cognitive psychology experiments.

**How long does it take you to find the red dot?**

## How to Play

- You have **10 levels** with increasing difficulty.
- Find the **single red circle** among many distractors (red squares, blue shapes, etc.).
- Click or tap the red dot as fast as possible.
- Each level has a time limit — beat the clock!

## Features

- Minimalist high-contrast design
- Progressive difficulty (more shapes, mixed colors & shapes)
- Local high score tracking
- Pause & Home buttons
- Fully playable on mobile and desktop
